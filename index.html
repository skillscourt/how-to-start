Welcome to SkillsCourt project.

To help with development please checkout vagrant:


$ git clone https://github.com/lukezbikowski/skillscourt-vagrant
$ cd skillscourt-vagrant
$ vagrant up

you should have your dev environment ready to go in few minutes.
