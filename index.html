Quick-start guide for developers 


          $ git clone https://github.com/lukezbikowski/skillscourt-vagrant
          $ cd skillscourt-vagrant
          $ vagrant up

Get yourself Yerba Mate (or just a coffee).
Your environment will be ready in few minutes.

What's happening now ?

  • Vagrant is getting you Ubuntu 14.04 LTS w/ git and rbenv
  • Installing ruby 2.1.2
  • checking out our github project
As ruby is build from source it might take a while...

What's next

Your code is in VAGRANT_DIRECTORY

Just bundle it!


          $ vagrant ssh
          $ cd /vagrant/skillscourt-api
          $ bundle install

Then startup the server


          $ bin/rails s
          $ cd /vagrant/skillscourt-api
          $ bundle install

The server is redirected to your host on 3030

Open localhost:3030 in your favourite browser!

Hosts configuration

To access our api you need to call subdomain api. To make it happen you need to configure your hostname (Windows, MacOS, Linux) or use pow (MacOS), prax (Linux)
